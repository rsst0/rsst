  1. Данный блог может быть в любой момент заблокирован сами-знаете-кем. Поэтому мы разделили файл-просмотрщик блога (который у вас сейчас открыт) и файл с содержимым блога (который подгружается автоматически). Для того, чтобы не потерять этот блог, настоятельно рекомендуем вам сохранить файл-просмотрщик блога локально на вашем компьютере или телефоне.
  2. Для сохранения файла-просмотрщика блога скачайте его по ссылке, переименуйте как вам удобно (для конспирации), сохранив расширение .html, и в дальнейшем открывайте этот файл локально на своем компьютере или телефоне. Содержимое блога будет автоматически подгружаться из другого файла, размещенного в интернете.
  3. При попытках сами-знаете-кого заблокировать содержимое блога мы будем по мере необходимости менять адрес размещения содержимого; новый адрес файлом-просмотрщиком будет определяться автоматически. Из-за смены адреса содержимого блога он может быть недоступен несколько дней, затем работоспособность блога восстановится. Файл-просмотрщик, сохраненный локально, при этом изменять не требуется.
  4. Сохраните значение md5-суммы файла-просмотрщика. Если по какой-либо причине вам придется скачать и локально сохранить файл-просмотрщик повторно, сверьте значение md5-суммы с ранее сохраненным; расхождение значений будет говорить о том, что файл подменен (нам противостоит очень серьезный противник и подмена нашего блога с целью дезинформации весьма вероятна).
